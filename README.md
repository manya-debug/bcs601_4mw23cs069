# Cloud Project - BCS601-4MW23CS069

## Project Name:
String Utility Web Application

## Description:
This project is a simple web application developed using Python Flask. It performs basic string operations such as reversing a string, counting vowels, and converting the string to uppercase. The application is deployed on a cloud platform.

## Technologies Used:
- Python
- Flask
- Gunicorn
- Cloud Shell
- GitHub
- Render (Cloud Platform)

## Steps to Implement:
1. Created project folder in Cloud Shell.
2. Developed the Python Flask application using vi editor.
3. Added required dependencies in requirements.txt.
4. Initialized Git and pushed code to GitHub repository.
5. Deployed the application using Render cloud platform.
6. Obtained the live URL after successful deployment.

## Output:
The application displays:
- Original string
- Reversed string
- Vowel count
- Uppercase string
