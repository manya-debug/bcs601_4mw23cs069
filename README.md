# Cloud Project - BCS601-4MW23CS069

## Project Name:
Multi Utility Calculator

## Description:
This project is a simple cloud-based web application developed to perform multiple mathematical operations online. The application provides users with different utility functions such as:
HCF & LCM Calculation
Reverse String
Factorial Calculation

## Technologies Used:
- Python
- Flask
- Gunicorn
- Cloud Shell
- GitHub
- Render (Cloud Platform)

## Steps to Implement:
1. Created project folder in Cloud Shell.
2. Developed the Python Flask application using vi editor.
3. Added required dependencies in requirements.txt.
4. Initialized Git and pushed code to GitHub repository.
5. Deployed the application using Render cloud platform.
6. Obtained the live URL after successful deployment.

## Output:
The application displays:
- Select the required operation:
-HCF & LCM
-Reverse String
-Factorial
-Enter the required input values.
-Click the Calculate/Reverse button.
-The application processes the data.
-Result is displayed instantly on the screen.
-Users can perform multiple operations online easily.
